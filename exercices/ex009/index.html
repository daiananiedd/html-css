Trabalhando com listas

Listas ordenadas

Para fazer listas ordenanas é só usar a tag OL e, dentro desta, a tag LI para cada item da lista.

  1. Acordar
  2. Ligar para João
  3. Tomar café
  4. Executar tarefas
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Ir para casa
  9. Jantar
  10. Dormir

Listas não ordenadas

Para listas não ordenadas a única diferença é que usamos a tag UL

  • Pão
  • Leite
  • Arroz
  • Mel
